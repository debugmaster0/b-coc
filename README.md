# Blockchain-Based Digital Evidence Chain of Custody

A Python proof-of-concept demonstrating how blockchain technology can be
integrated into digital forensic software to create a secure digital chain
of custody.

## Features

- Multi-chain EvidenceLedger
- SHA-256 block hashing
- ECDSA digital signatures
- Ledger state root validation
- Independent chain validation
- PySide6 graphical interface
- Evidence transfer simulation
